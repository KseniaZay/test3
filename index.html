Git Hub

  1. git --version (Проверка версии GIT)
  2. git config --global user.name (имя пользователя)
  3. git config --global user.email (email пользователя)
  4. git config --list (показать все настройки,)
  5. ssh-keygen -t ed25519 -C (Вводим email который указан на GitHub аккаунте)
  6. git status (Отображение текущего состояния GIT репозитория.)
  7. cd (переход по папкам)
  8. git config key (проверить значение конкретного ключа)
  9. git init (Создание нового репозитория)
  10. git clone (Клонирование репозитория)
